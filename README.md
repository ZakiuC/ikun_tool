# Ikun 工具

监听键盘按键并做出反应

## 使用方法
### 自定义图片
    在D盘创建一个ikun文件夹，放入名为0.jpg和1.jpg的图片，可以代替闭嘴和张嘴时的图片

### 自定义音频
    在D盘创建一个ikun文件夹，放入名为a.mp3的音频，可以在摁下“a”的时候播放；放入名为c_a.mp3的音频，则可以在摁下“CTRL+A”快捷键的时候播放。
